sod

Secure-Enclave-backed SSH for macOS. Your authentication key is generated inside the Secure Enclave and never leaves it; Touch ID is required for every signature.

This installer places the sod command-line tool in /usr/local/bin and its manual page in /usr/local/share/man/man1.

Nothing else is changed: no login items, no shell configuration, and no SSH_AUTH_SOCK changes. Those steps are opt-in and shown when the install finishes.
