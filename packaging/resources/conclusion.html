sod is installed

sod is in /usr/local/bin. Get started:

sod ssh-keygen -f ~/.ssh/id_sod
eval "$(sod ssh-agent)"
sod ssh-add ~/.ssh/id_sod
ssh -i ~/.ssh/id_sod user@host

To start the agent automatically at login (optional):

sod ssh-agent --install-launch-agent
# then add to your shell profile:
export SSH_AUTH_SOCK="$HOME/.ssh/sod-agent.sock"

See man sod for full documentation.
