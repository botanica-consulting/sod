sod is installed

sod is in /usr/local/bin. Get started:

sod ssh-keygen     # create ~/.ssh/id_sod
sod install        # run the agent at login + print the line for your shell
sod ssh-add        # load the key

sod install prints one SSH_AUTH_SOCK line to paste into your shell startup file (~/.zshrc, ~/.bash_profile, config.fish, …). Open a new shell, then ssh as usual — Touch ID prompts on each connection.

See man sod for full documentation.
